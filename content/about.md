## LLM 다이어리

이 블로그는 **로컬 LLM** 과 함께 브레인스토밍하고, 그 과정을 Wiki 로 관리하는 공간입니다.

### 목표
- 로컬 LLM 의 지식을 Wiki 로 축적
- 반복적인 문제 해결 패턴 기록
- 기술 결정 사항 (ADR) 관리
- pi 에이전트와 연동하여 자동화

### 기술 스택
- **Hugo** - 정적 사이트 생성기
- **PaperMod** - 테마
- **Git** - 버전 관리
- **pi 에이전트** - Markdown 자동 생성

###Wiki 구조
```
content/wiki/
├── troubleshooting/   # 오류 해결
├── decisions/# 아키텍처 결정
├── tools/    # 도구 사용법
└── projects/ # 프로젝트별
```
