## 프로젝트 개요

이 블로그는 **Hugo**로 구축된 정적 블로그 사이트입니다.  
LLM(Large Language Model) 관련 학습 내용, 오류 해결 기록, 기술 결정 사항을 정리하는 다이어리 형식의 사이트입니다.

## 폴더 구조

```
llm-diary/
├── content/
│   ├── posts/        # 블로그 포스트 (Markdown)
│   └── wiki/         # LLM Wiki (Markdown)
│       ├── errors/   # 오류 해결 기록
│       ├── decisions/# 아키텍처 결정 사항
│       ├── tools/    # 도구 사용법
│       └── projects/ # 프로젝트별 기록
├── themes/PaperMod/  # PaperMod 테마 (git submodule)
├── hugo.toml         # Hugo 설정 파일
├── assets/           # 정적 자산
├── static/           # 정적 파일
├── layouts/          # 커스텀 레이아웃
├── data/             # 데이터 파일
├── i18n/             # 국제화 파일
└── archetypes/       # 새 포스트 템플릿
```

## 주요 특징

- **Hugo** 기반의 빠른 정적 사이트 생성
- **PaperMod** 테마 적용
- Wiki 형식으로 지식 관리 (오류 해결, 기술 결정, 도구 사용법 등)
- Git을 통한 버전 관리

## 로컬 테스트

```bash
hugo server -D --buildFuture --bind 0.0.0.0 --port 1313
```
