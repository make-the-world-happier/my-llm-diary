## 오늘 나눈 이야기 요약

"노는 컴퓨터로 뭘 할 수 있을까?"라는 질문에서 시작된 브레인스토밍입니다. 비트코인 채굴부터 시작해 OpenClaw + Ollama 기반 로컬 AI 에이전트 구축, 그리고 구체적인 데이터 수집 전략까지 논의했습니다.

## 핵심 질문

> "맥북을 포함해 노는 노트북이 몇 대 있는데, 이걸로 뭘 할 수 있는게 없을까?"

## 결론부터 말하면

**임대형 수익화 (Salad, Vast.ai 등)는 전기세 차감 후 거의 무의미하지만, OpenClaw + Ollama 기반 로컬 AI 에이전트 구축은 장기적으로 충분히 가치 있는 투자입니다.**

## 1️⃣ 비트코인 채굴: 개인에게 의미가 있을까? (2026년 기준)

### 결론: **거의 의미 없음**

비트코인 채굴은 이제 "취미"가 아니라 "산업"입니다.

| 요소 | 현재 상황 |
|------|-----------|
| **전기 요금** | $0.06~0.07/kWh 이하가 필수 |
| **장비 효율** | 15~16 J/TH 이하의 최신 ASIC 필요 |
| **블록 보상** | 2024년 반감기 이후 **3.125 BTC** |
| **네트워크 난이도** | 역사적 최고치 근처, 계속 상승 |

### 개인 채굴이 어려운 이유

1. **전기 요금 장벽** — 가정용 전기료로는 채굴기 가동비만으로도 손실
2. **장비 비용** — 최신 ASIC 한 대에 수천~수만 달러
3. **채굴 풀 의존** — 혼자서는 블록을 찾을 확률이 거의 0
4. **반감기 효과** — 2024년 4월 반감기로 보상 절반

### 현재 우려되는 신호

- 2026년 4월 기준, **채굴업체들의 BTC 매도 지속** — 수익성 악화로 인한 공급 압력
- BTC 가격이 $5,000~$6,000 아래로 떨어지면 많은 채굴기가 적자
- 일부 채굴기 운영자가 BTC를 버리고 다른 코인으로 전환하는 움직임

## 2️⃣ 맥북으로 다른 코인 채굴: 가능한가?

### 결론: **아니요. 거의 불가능합니다.**

| 문제점 | 설명 |
|--------|------|
| **Apple Silicon은 채굴용이 아님** | M1~M4 모든 칩은 채굴 알고리즘에 최적화되지 않음 |
| **발열 문제** | 장시간 풀가동 시 스로틀링 발생 → 성능 급락 |
| **배터리 손상** | 상시 풀가동 = 배터리 팽창·수명 단축 |

### Monero (XMR) — CPU 채굴이 가능한 거의 유일한 코인

| CPU | 해시레이트 | 일일 수익 |
|-----|-----------|----------|
| 최상위 데스크탑 (Ryzen 9 7950X) | 25~28 KH/s | 약 $0.10~0.15 |
| 맥북 M1 Pro | 약 3~5 KH/s | 약 $0.01~0.02 (1~2원) |
| 맥북 M3 Pro | 약 5~7 KH/s | 약 $0.02~0.03 (2~3원) |

> 하루에 1~3원입니다. 배터리 손상분을 빼면 **순수익은 마이너스**.

Reddit에서 M3 Pro 채굴 테스트한 사람의 결론:
> *"자신이 채굴하는 것보다 유튜브에서 남들이 채굴하는 거 보는 게 더 수익적이다."*

## 3️⃣ 노는 컴퓨터로 할 수 있는 것들 (2026년 기준)

### 3-1. AI 컴퓨팅 임대 (가장 유망하지만)

AI 붐으로 **사용자의 GPU/CPU를 빌려주는 플랫폼**이 활발해졌습니다.

| 플랫폼 | 설명 | 특징 |
|--------|------|------|
| **Salad** | 가장 대중적. 45만+ 사용자 | Windows 전용. 설치만 하면 자동 실행 |
| **Vast.ai** | GPU 렌탈 마켓플레이스 | RTX 4090급이면 시간당 $0.30~0.60 |
| **Aethir** | 분산 GPU 클라우드 | Windows. GPU 임대 → AI/렌더링 작업 |
| **GamerHash** | 80만+ 사용자. AI 추론용 | Windows. 비교적 쉽게 시작 가능 |
| **Hyperlink** | CPU+GPU 임대. 은행 계좌 직접 입금 | 크립토 불필요 |

#### 예상 수익 (Windows GPU 노트북 기준)

| GPU | 시간당 수익 | 일일 (24h) | 월간 (24h) |
|-----|-----------|-----------|-----------|
| RTX 3060 (6GB) | $0.02~0.05 | $0.50~1.20 | $15~35 |
| RTX 4070 | $0.06~0.10 | $1.50~2.40 | $45~70 |
| RTX 4090 | $0.30~0.60 | $7.00~14.00 | $210~420 |

> ⚠️ **전기세 차감 필수**. 한국 전기세 기준, RTX 3060급 노트북(150W) 기준으로 월 $15 수익 중 약 $10~15가 전기세. **순수익 거의 0에 가까움**.

### 3-2. 분산 렌더링

3D 렌더링 작업을 분산해서 수행하고 보상을 받는 방식.

| 플랫폼 | 설명 |
|--------|------|
| **Render Network (RNDR)** | 분산 렌더링 네트워크. GPU 필요 |
| **LuxCoreRender** | 오픈소스 렌더링. 크립토 보상으로 지급 |

### 3-3. 분산 스토리지 임대

저장 공간을 빌려주고 보상을 받는 방식.

| 플랫폼 | 설명 |
|--------|------|
| **Storj** | 분산 클라우드 스토리지. HDD/SSD 용량 필요 |
| **Filecoin** | 블록체인 기반 분산 스토리지 |
| **Sia** | 탈중앙화 스토리지 네트워크 |

> 노트북 내장 저장공간으로 보면 월 $1~5 수준. 거의 의미 없음.

### 3-4. 분산 네트워크 대역폭 임대

인터넷 연결을 공유하고 보상을 받는 방식.

| 플랫폼 | 설명 |
|--------|------|
| **Honeygain** | 가장 대중적. 월 $5~20 (대역폭에 따라) |
| **Pawns.app** | Honeygain과 유사 |
| **PacketStream** | 프록시 네트워크 대역폭 임대 |

> ⚠️ **보안 주의**. 타인의 트래픽이 내 네트워크를 통과하므로 개인정보 노출 리스크가 있습니다.

### 3-5. 과학/연구 분산 컴퓨팅 (무보상 but 의미 있음)

| 프로젝트 | 설명 |
|----------|------|
| **BOINC** | Fold@home(단백질 접힘), SETI(외계 신호) 등 |
| **World Community Grid** | 질병 연구, 청정에너지 등 |

> 돈은 안 되지만, 인류에 기여한다는 의미에서 좋습니다.

## 4️⃣ 임대 말고 AI 에이전트 구축: OpenClaw + Ollama

### 💡 이 방식이 가장 실용적입니다

임대형 수익화와 달리, **자신만의 AI 에이전트 인프라**를 구축하는 건 장기적으로 가치가 큽니다.

#### 📐 아키텍처

```
[OpenClaw (에이전트 오케스트레이터)]
         ↕ API
[Ollama (로컬 LLM 서빙)]
         ↕
[로컬 모델: Llama 3.2, Qwen 2.5, Gemma 3 등]
         ↕
[도구: 웹 검색, 파일 읽기/쓰기, 브라우저 제어, 터미널]
```

#### ✅ 할 수 있는 일

| 작업 | 설명 |
|------|------|
| **웹 데이터 수집** | OpenClaw의 `web_search` 프로바이더로 검색 + 결과 분석 |
| **자동 리포트 작성** | CSV/JSON 데이터 → 요약 리포트 → 이메일 발송 |
| **파일 분석** | 로컬 파일 읽어서 패턴 분석, 이상치 탐지 |
| **브라우저 자동화** | 웹 스크래핑 (로그인, 폼 입력, 데이터 추출) |
| **定时 작업** | cron 또는 OpenClaw 스케줄러로 자동 실행 |
| **메시지 연동** | Discord/Telegram/Slack으로 에이전트 제어 |

#### 💰 비용: **완전 무료**

- OpenClaw: 오픈소스
- Ollama: 오픈소스
- Llama 3.2 / Qwen 2.5 / Gemma 3: 모두 오픈웨이트
- API 호출비: 0원 (모두 로컬)

## 6️⃣ 어떤 데이터를 수집할 것인가?

### 🔥 Tier 1: 가장 실용적이고 고부가가치

#### 1. DART 공시 분석 (한국 주식)

| 항목 | 수집원 | 설명 |
|------|--------|------|
| 기업 공시 | DART (dart.fss.or.kr) | 공시 요약 → 인사이트 추출 |
| 기관/외국인 동향 | 증권사 실시간 데이터 | 수급 분석, 이상 징후 탐지 |
| 경제 지표 | 한국은행 통계, 통계청 | 금리, 물가, 고용 데이터 |

**워크플로우:**
```
매일 07:00 → DART 공시 크롤링 → M1 Max에서 요약 → 이상 공시 알림
매일 16:00 → 시장 마감 데이터 → 포트폴리오 리포트 생성
매주 월요일 → 기관 수급 TOP/BOTTOM → 분석 리포트
```

| 항목 | 수집원 | 설명 |
